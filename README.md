# gravity-simulator
This is a project where I created a simple gravity simulator using NASM x64 assembly files. This is a project that I did for a few weeks where using my NASM x64 knowledge, I created a physics game where there were rocks that needed to be counted and using x64 assembly, it would read out how manyy rocks there were within the simulator.
